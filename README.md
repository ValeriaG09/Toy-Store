TOY STORE

DESCRIPCIÓN:
Toy Store es una plataforma enfocada en la venta de productos para adultos (sex shop), diseñada para ofrecer una experiencia moderna, segura y discreta a los usuarios.	
El proyecto busca integrar un catálogo atractivo, navegación intuitiva y un sistema eficiente de gestión de productos. Este sistema está pensado tanto para clientes que desean comprar productos como 
para administradores que necesitan gestionar inventario, precios y pedidos.

OBJETIVOS DEL PROYECTO:
1. Crear una tienda online funcional y atractiva
2. Ofrecer una experiencia de usuario intuitiva
3. Implementar un sistema de gestión de productos
4. Garantizar privacidad y seguridad en la navegación
5. Facilitar la administración del negocio

FUNCIONALIDADES:
- USUARIO - 
1. Registro e inicio de sesión
2. Navegación por catálogo de productos
3. Visualización de detalles de productos
4. Agregar productos al carrito
5. Proceso de compra

- ADMINISTRADOR -
1. Gestión de productos (crear, editar, eliminar)
2. Control de inventario
3. Gestión de pedidos
4. Administración de precios

TECNOLOGIAS UTILIZADAS:
1. Frontend: HTML, CSS, JavaScript
2. Backend: Node.js
3. Base de datos: Mysql
4. Control de versiones: Git & GitHub
